## 🖥️ What L0p4Map Does

L0p4Map helps you see devices and services on a network in a clear way. It scans a local network, checks IP addresses, and builds a live map of what it finds. You can also point it at a custom IP or website for a more focused check.

It is built for Windows users who want a simple way to look at network activity without reading complex logs. The app combines:

- Fast ARP-based device discovery
- Nmap support for deeper checks
- Live network topology view
- Port and host details in one place

## 🧭 Main Features

L0p4Map uses ARP discovery to find devices on a local network. This helps it spot active devices without long waits.

The topology engine turns scan results into a visual map. This makes it easier to see which devices are present and how they relate.

### 🧪 Nmap Integration
The app uses nmap for deeper inspection. That helps when you want port and host details beyond a simple device list.

You can check which ports are open on a device or site. This helps you understand what services may be reachable.

### 🧰 Flexible Targets
You can scan a whole network, a single IP, or a website host. That gives you a simple path for both broad and focused checks.

### 🪟 Windows-Friendly Interface
The app is built for regular Windows use. You do not need to work from a command line to begin using it.

## ❓ Troubleshooting

### The App Does Not Open
- Make sure you downloaded the latest release
- Check that the file finished downloading
- Try extracting the ZIP file again
- Move the file to a simple folder like Downloads

### No Devices Appear
- Check that you picked the right network range
- Make sure you are connected to the network you want to scan
- Try again after a short wait
- Test a single IP address first

### The Scan Is Slow
- Large networks take longer to scan
- Wi-Fi can be slower than wired connections
- Close other apps that use a lot of memory
- Try a smaller target range

### Results Look Empty
- Some hosts block scan replies
- A website may hide parts of its setup
- Try a different target or a direct IP address
- Run the scan again after checking the target format
